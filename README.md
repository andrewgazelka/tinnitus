```
A TRT app for audio notching

Usage: tinnitus [OPTIONS] <FREQUENCY> [Q]

Arguments:
  <FREQUENCY>  frequency of the tinnitus
  [Q]          Q of the filter [default: 1000.0]

Options:
  -v, --volume <VOLUME>  volume to start at [default: 0.1]
  -h, --help             Print help
  -V, --version          Print version
```
